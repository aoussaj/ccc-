
Electronic proporties_parsing: This script extracts electronic excited-state properties from the STEOM-CCSD RESULTS section of an ORCA output file.

Duschinsky_matrix_plot.py: Python script to load and visualize the Duschinsky matrix obtained from FCClasses3 calculations.
The plot shows the correlation between ground-state (GS) and excited-state (ES) vibrational modes of a BODIPY dye. Mode pairs with strong mixing are automatically detected using a threshold and highlighted on the matrix.
